# CivicConnect - Ghaziabad Civic Issue Tracker

**🏠 From Complaint to Accountability - Hyperlocal Civic Platform for Ghaziabad**

CivicConnect is a **hackathon-winning platform** that transforms chaotic WhatsApp complaints into **structured, trackable civic action** for Ghaziabad sectors (Vaishali, Indirapuram, Raj Nagar Extension, etc.).

## ✨ **Demo Features**

- **📍 8 Ghaziabad Localities**: Vaishali, Indirapuram, Raj Nagar, Vasundhara + more
- **⚡ Real-time Issue Tracking**: Reported → Verified → In Progress → Resolved  
- **📊 Hall of Shame Dashboard**: Public accountability for RWAs/Councillors
- **👥 User Authentication**: Login/Register for residents
- **🔧 Admin Panel**: Status updates + issue management
- **📱 Mobile Responsive**: Beige off-white + brown theme

## 🎯 **Hackathon Innovation**

```
Unlike E-Samadhan (govt bureaucracy) or WhatsApp groups (no tracking):
✅ Forces RWA accountability via public Hall of Shame
✅ Sector-wise performance metrics (Indirapuram worst, Vaishali best)  
✅ Revenue-ready: Housing society SaaS + civic data sales
✅ Zero hardware: Smartphone-only deployment
```

## 🏃‍♂️ **Local Setup** (5 minutes)

```bash
# 1. Clone & Environment
git clone https://github.com/YOUR-USERNAME/civicconnect.git
cd civicconnect

# 2. Virtual Environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate  # Windows

# 3. Install Dependencies
pip install -r requirements.txt

# 4. Load Demo Data
python demo_data.py

# 5. Run Server
python app.py
```

**Visit:** `http://localhost:5000`
